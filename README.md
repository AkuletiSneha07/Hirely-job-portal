# Job Portal

A Flask-based job portal application.

## Setup

1.  Clone the repository.
2.  Create a virtual environment: `python -m venv venv`
3.  Activate the virtual environment: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (macOS/Linux)
4.  Install dependencies: `pip install -r requirements.txt`
5.  Create a `.env` file with your database credentials and secret key.
6.  Create the database in MySQL.
7.  Run the application: `python app.py`

## Usage

1.  Register as a job seeker or employer.
2.  Log in.
3.  Browse job listings or post a job.
4.  Apply for jobs.
5.  Admin users can manage users and job listings.
